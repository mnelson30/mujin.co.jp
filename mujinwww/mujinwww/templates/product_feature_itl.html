{% trans "Industrial Task Language is a robot programming language developed by MUJIN, which enables users to conveniently program and design on their robotic applications. ITL is completely indenpendent to any robot manufactures' and designed to maximize MUJIN Controller's extensive simulation features. {% trans "Specify only goals" %}

{% trans "Since MUJIN Controller is the tool creating the most optimized path consisted of avoidance points, in ITL user is supposed to specify only goals." %}
