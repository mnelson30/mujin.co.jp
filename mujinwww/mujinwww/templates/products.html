{% trans "The World's First" %} {% trans " Multi-Robot Manipulation Planning System" %}

{% trans "Most other robot simulation packages consider just a single robot and are not configured to compute solutions for multiple robots simultaneously. Because of this handicap, robot programmers have a very difficult time producing non-colliding motions for all robots. MUJIN has developed algorithms to make collision-free motion plans for multiple robots operating simultaneously. We also make it possible to optimize the combined locations of all robots involved in the task. The Multi-robot functionality would allow customers to use multiple robots together in an assembly line, which can result in a substantial gain in productivity. We can also optimize for cell-manufacturing scenarios where a dual-arm robot performs a complex assembly task." %}

{% trans "The World's First" %}{% trans " Parallelized Placement Optimization System" %}

{% trans "What are the feasible locations of the robot where the task can be completed? Among these feasible locations, which allows the robot to perform the task most quickly? To answer these questions, we need to evaluate thousands of robot positions and orientations, and for each robot placement we need to solve for an optimal motion plan. For existing automation tools, it may take too much time to reasonably perform such optimization while maintaining task and safety constraints. MUJIN Controller is designed with the latest parallel computing technology that enables us to provide a good solution in a relatively short period of time while taking into account collision avoidance, singularities, and sensor-based visibility constraints." %}
