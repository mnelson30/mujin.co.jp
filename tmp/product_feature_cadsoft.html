Standalone 3D CAD Format Support

Many of existing robot simulation software are implemented as add-on's to expensive 3D CAD software. Purchasing new 3D CAD software licenses can be expensive. The MUJIN Controller does not require any specific 3D CAD software and the robot programmer can easily import 3D models in most common formats.

Supported formats:

  • COLLADA
  • IGES
  • STEP
  • IV/VRML
  • STL
  • DirectX
