    Introduction of MUJIN Controller


    Simulation Precision and Accuracy

    A simulation is pointless unless it is consistent with the reality. Most existing simulation tools have difficulty with ensuring that a planned trajectory matches up with the robot's motion in reality. MUJIN however is proud to guarantee our trajectories to at least 95%% precision. What the program shows on the computer is what will happen in the actual robot cell.


    Product Features


    Inverse Kinematics

    The IK solver can find analytical solutions to your robot's IK, even for more than 7axis machine.

    Collision Avoidance

    The collision avoidance automatically finds efficient paths without specifying intermediate waypoints.

    Simulation Precision

    Trajectories are guaranteed to be at least 95%% accurate when compared to the motion of the real robot.

    Grasp/Release Simulation

    Simulate trajectories while considering a workpiece in the robots grasp

    Grasp Planning

    Planning the most suited grasps out of specified grasp sets for the task.

    Placement Optimizations

    Optimize your layout by examining all possible positions and trajectories in an environment.

    Clearance Setting

    It is possible to set an obstacle clearance so that all parts of the robot and its grabbed objects can be at least specified distance away from obstacles.

    Multi-Robot Manipulation

    MUJIN leverages planning algorithms for collision-free motion plans with multiple robots in the same environment.

    Industrial Task Language

    ITL is MUJIN's easy to learn, generic robot programming language which can specify tasks for any robot manufacturer.

    Robot Program Output

    The controller outputs the robot manufacturer's program, so there is no need to write it yourself.

    Zero Installation

    Use the controller through your web browser on any platform you like.

    No CAD Dependencies

    The controller does not require any specific 3D CAD software. Users can easily import from most common formats.
